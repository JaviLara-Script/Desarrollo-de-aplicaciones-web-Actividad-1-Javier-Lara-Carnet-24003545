# ToDo App - React Bootstrap

Aplicación web simple para gestionar tareas y metas personales.

## Tecnologías

- React
- React Bootstrap
- CSS

## Cómo usar

1. Clona el repositorio.
2. Ejecuta `npm install`.
3. Ejecuta `npm start`.

También puedes probarla en [PlayCode](https://playcode.io/react).
